# Blink

Automated eye-break reminders for macOS — the 20-20-20 rule, on autopilot.

Every 20 minutes, Blink fades in a calm full-screen breathing overlay reminding
you to look 20 feet away for 20 seconds. Lives in your menu bar. No accounts,
no telemetry, all local.

- **Free:** automatic 20-minute micro-breaks, breathing overlay across all displays.
- **Pro ($19, one-time):** custom schedules, long stretch breaks, work hours,
  strict mode, daily stats & streaks.

Website: https://blink.vercel.app
